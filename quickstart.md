# Quick Start

BotMate is an open-source platform that allows you to create, manage, and deploy Telegram bots with ease. With its intuitive tools and user-friendly interface, even those with little to no coding experience can create functional bots in no time. And when it comes to managing your bots, BotMate has got you covered with a beautifully designed web-based dashboard that lets you easily monitor and control all of your bots in one place. BotMate is built with scalability in mind, meaning it can handle a large number of bots with ease.

BotMate can be easily deployed using Docker, making it a breeze to get started. Whether you're looking to create bots for personal or professional use, BotMate is the perfect solution for your needs.

## Installations

There are multiple ways to setup BotMate server. You can either use Docker or install it manually form source code.
