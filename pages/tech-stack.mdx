# Tech Stack

Here are the technologies used in this project.

## Frontend

React is used for the front end of this project. It is a JavaScript library for building user interfaces.
Along with React, we use the following libraries:

- [React Router](https://reactrouter.com/) - for routing
- [Chakra UI](https://chakra-ui.com/) - for styling
- [React Query](https://react-query.tanstack.com/) - for data fetching
- [React Hook Form](https://react-hook-form.com/) - for form handling
- [React Icons](https://react-icons.github.io/react-icons/) - for icons

## Backend

Express is used for the back end of this project. It is a web application framework for Node.js.
Along with Express, we use the following libraries:

- [Mongoose](https://mongoosejs.com/) - for MongoDB object modeling
- [Bcrypt](https://www.npmjs.com/package/bcrypt) - for password hashing
- [Json Web Token](https://www.npmjs.com/package/jsonwebtoken) - for authentication
- [Dotenv](https://www.npmjs.com/package/dotenv) - for environment variables

## Others

- [TypeSCript](https://www.typescriptlang.org/) - for type checking
- [Vite](https://vitejs.dev/) - for bundling
- [ESLint](https://eslint.org/) - for linting
- [Prettier](https://prettier.io/) - for code formatting
- [MongoDB](https://www.mongodb.com/) - for database
- [vm2](https://www.npmjs.com/package/vm2) - for sandboxing
